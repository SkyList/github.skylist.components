# SkyList Components #

## Biblioteca de web components compativel com qualquer sistema web ##

https://www.npmjs.com/package/@skylist/components
---------

## Quick Start
instale os pacotes

```sh
$ npm install
```
ou
```sh
$ yarn
```

## desenvolvimento
Para abrir o preview com Storybook com hot reload
```sh
$ npm start
```

ou

```sh
$ yarn start
```

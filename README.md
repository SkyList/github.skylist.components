# SkyList Components #

## Biblioteca de web components compativel com qualquer sistema web ##

https://www.npmjs.com/package/@skylist/components
---------

## Quick Start
instale os pacotes

```sh
$ npm install
```
ou
```sh
$ yarn
```

## Preview com storybook
```sh
$ npm start
```
ou
```sh
$ yarn start
```

## desenvolvimento
para desenvolver, basta rodar os comandos em terminais diferentes
```sh
$ npm start
```
em outro terminal:
```sh
npm run build-watch
```
